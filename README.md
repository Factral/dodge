## Introduction

**#Dodge** is a minigame in which you must survive the cannons that are thrown at you, you can play with friends and try to see who is the last to die, also, you can see how long you manage to survive the cannons!

## How to Play?

You can run this game in #castle rooms _(`/room #castle0dodge`)_

or
 
<b>open the file <a href="https://github.com/Factral/dodge/blob/main/dodge.lua">dodge.lua</a> and copy the code to run it in a tribe house.</b>
